Component RingBuffer
Implements a ring buffer for 16bit elements
Component Level: High
Category: SW-User Components
Methods:
(Methods are user-callable functions/subroutines intended for the component functions control. Please see the Embedded Components page for more information.)
  • Clear - Clear (empty) the ring buffer.

      ANSIC prototype: void Clear(void)

  • Put - Puts a new element into the buffer

      ANSIC prototype: uint8_t Put(ComponentName_ElementType elem)
    • elem:ComponentName_ElementType - New element to be put into the buffer
    • Return value:uint8_t - Error code

  • Putn - Put a number new element into the buffer.

      ANSIC prototype: uint8_t Putn(ComponentName_ElementType *elem, ComponentName_BufSizeType nof)
    • elem: Pointer to ComponentName_ElementType - Pointer to new elements to be put into the buffer
    • nof:ComponentName_BufSizeType - number of elements
    • Return value:uint8_t - Error code

  • Get - Removes an element from the buffer

      ANSIC prototype: uint8_t Get(ComponentName_ElementType *elemP)
    • elemP: Pointer to ComponentName_ElementType - Pointer to where to store the received element
    • Return value:uint8_t - Error code

  • Peek - Returns an element of the buffer without removiing it.

      ANSIC prototype: uint8_t Peek(ComponentName_BufSizeType index, ComponentName_ElementType *elemP)
    • index:ComponentName_BufSizeType - Index of element. 0 peeks the top element, 1 the next, and so on.
    • elemP: Pointer to ComponentName_ElementType - Pointer to where to store the received element
    • Return value:uint8_t - Error code

  • Compare - Compares the elements in the buffer.

      ANSIC prototype: uint8_t Compare(ComponentName_BufSizeType index, ComponentName_ElementType *elemP, ComponentName_BufSizeType nof)
    • index:ComponentName_BufSizeType - Index of element. 0 peeks the top element, 1 the next, and so on.
    • elemP: Pointer to ComponentName_ElementType - Pointer to elements to compare with
    • nof:ComponentName_BufSizeType - number of elements to compare
    • Return value:uint8_t - zero if elements are the same, -1 otherwise

  • Delete - Removes an element from the buffer

      ANSIC prototype: uint8_t Delete(void)
    • Return value:uint8_t - Error code

  • NofElements - Returns the actual number of elements in the buffer.

      ANSIC prototype: ComponentName_BufSizeType NofElements(void)
    • Return value:ComponentName_BufSizeType - Number of elements in the buffer.

  • NofFreeElements - Returns the actual number of free elements/space in the buffer.

      ANSIC prototype: ComponentName_BufSizeType NofFreeElements(void)
    • Return value:ComponentName_BufSizeType - Number of elements in the buffer.

  • Deinit - Driver de-initialization

      ANSIC prototype: void Deinit(void)

  • Init - Initializes the data structure

      ANSIC prototype: void Init(void)


Note: Some methods can be implemented as macros.
