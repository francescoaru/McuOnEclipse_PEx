Component RingBuffer
Implements a ring buffer for 16bit elements
Component Level: High
Category: SW-User Components
Properties:
(Properties are parameters of the component. Please see the Embedded Components page for more information.)
  • Component name - Name of the component.
  • SDK - Link to SDK
  • Reentrant - If the driver shall be reentrant. If set to 'yes', then a critical section is used to access the global ring buffer data.
  • Critical Section - Interface to critical section handling
  • Buffer Size - Number of elements for the buffer. If using < 256 bytes, 8bit index type is used, otherwise a 16bit one.
  • Element Size - Size of a single element in the ring buffer

    There are 3 options:
    • 1 Byte: 1 Byte
    • 2 Bytes: 2 Bytes
    • 4 Bytes: 4 Bytes
