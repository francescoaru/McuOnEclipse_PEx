Component GenericBitIO
Generic BitIO Bean
Component Level: Low
Category: CPU Internal Peripherals-Port I/O
Properties:
(Properties are parameters of the component. Please see the Embedded Components page for more information.)
  • Component name - Name of the component.
  • Direction - Direction of the bit
  • SDK - Link to Kinetis SDK
  • SDK - If enabled, it will us the Kinetis SDK API for low level bit operations.
    The following items are available only if the group is enabled (the value is "Enabled"):
    • BitIO - Implementation of BitIO with SDK
  • Data - Group defining how the data is accessed
    • Declaration - How the bit is declared.

      There are 5 options:
      • By Bit Name: Resource is defined with a symbol/name identifying the register bit.
      • By Port Name: Resource is defined with a symbol/name identifying the register plus a bit number.
      • By Address: Resource is defined with an address identifying the register plus a bit number.
      • By LDD: Ressource is using Logical Device Driver (LDD).
      • Set Clear Port Name: Ressource is using special set and clear registers.

    • By Bit Name - The resource is accessed using bit names
      The following items are available only if the group is enabled (the value is "Enabled"):
      • Name - Symbol name to access the bit data
    • By Port Name - The ressource is uing port name
      The following items are available only if the group is enabled (the value is "Enabled"):
      • Name - Port name to use
      • Bit Number - Bit number to clear/set in port name
    • By Set/Clear Port Name - Way to access bit using special bit set and clear registers
      The following items are available only if the group is enabled (the value is "Enabled"):
      • Neg Name - Port name to negate/toggle data
      • Clr Name - Port name to clear data
      • Set Name - Port name to set data
      • Get Name (input) - Port name to get input data if port is in input mode
      • Get Name (output) - Port name to get data if port is in output mode
      • Bit Number - Bit number in port
    • By Address - The ressource is accessed by its address.
      The following items are available only if the group is enabled (the value is "Enabled"):
      • Address - Address
      • Bit Number - Bit number
    • By LDD - The resource is accessed using Logical Device Driver
      The following items are available only if the group is enabled (the value is "Enabled"):
      • Field Name - Symbol name to access the bit data. This name is defined in the GPIO_LDD component.
      • GPIO LDD - Interface to the GPIO LDD component
  • Direction - Direction of the bit
    • Declaration - How the bit is declared

      There are 5 options:
      • By Bit Name: Resource is defined with a symbol/name identifying the register bit.
      • By Port Name: Resource is defined with a symbol/name identifying the register plus a bit number.
      • By Address: Resource is defined with an address identifying the register plus a bit number.
      • By LDD: Ressource is using Logical Device Driver (LDD).
      • Set Clear Port Name: Ressource is using special set and clear registers.

    • 1 means output - If a one bit means using the port as output pin (otherwise zero will be used for setting up the port as output).
    • By Bit Name - The resource is accessed using bit names
      The following items are available only if the group is enabled (the value is "Enabled"):
      • Name - Symbol name to access the bit data
    • By Port Name - The ressource is uing port name
      The following items are available only if the group is enabled (the value is "Enabled"):
      • Name - Name of the bit
      • Bit Number - Number of the bit
    • By Address - The ressource is accessed by its address.
      The following items are available only if the group is enabled (the value is "Enabled"):
      • Address - Address
      • Bit Number - Bit number
  • Initialization - Defines the initialization value of the bit.
    • Value - Initial value of the bit
    • Direction - Initial direction of the bit

      There are 2 options:
      • input: Port is configured as input.
      • output: Port is configured as output.
