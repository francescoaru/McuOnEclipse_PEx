Component QuadCounter
Counter/Decoder for a Digital Quadrature Encoder
Component Level: Low
Category: CPU External Devices-Motor Control
Properties:
(Properties are parameters of the component. Please see the Embedded Components page for more information.)
  • Component name - Name of the component.
  • C1 and C2 swapped - With this settings, the pins C1 and C2 are swapped. That way the tick counting can be reverted.
  • Counter Type - Type for quadrature counter

    There are 2 options:
    • 16bit: Quadcounter is 16bit
    • 32bit: Quadcounter is 32bit

  • Method - This group controls how the signals are aquired: either through sampling or with input capture.
    • Sampling - This group controls how the signals are sampled. If not enabled, then input capture is used.
      The following items are available only if the group is enabled (the value is "Enabled"):
      • Error Correction - If decoding shall be done with error correction.
      • C1 - Input C1 signal from the encoder.
      • C2 - Input C2 signal from the encoder.
    • Input Capture - This group controls the input capture
      The following items are available only if the group is enabled (the value is "Enabled"):
      • Number of steps per capture overflow event. - The number of steps for each input capture event.
      • C1 Capture - C1 signal from the encoder
      • C2 - Input C2 signal from the encoder.
  • Shell - If enabled, it adds command line shell support to the component
    The following items are available only if the group is enabled (the value is "Enabled"):
    • Shell - Interface to shell
    • Utility - Interface to utility functions used by shell
  • SDK - Link to SDK
