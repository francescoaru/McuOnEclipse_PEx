  • ReadEscapedName - Scans an escaped name from a string. This is useful e.g. for double quoted file names.

      ANSIC prototype: byte ReadEscapedName(const unsigned char *filename, uint8_t *destname, size_t maxlen, size_t *lenRead, size_t *lenWritten, const char *terminators)
    • filename: Pointer to uint8_t - the name to be copied. Names may be in quoted format
    • destname: Pointer to uint8_t - the destination of the copy. Names are not in quoted format. destname may be NULL to get the other return values only
    • maxlen:uint8_t - length allocated for destname
    • lenRead: Pointer to char - length read in filename to copy the whole name. Note that filenames maybe space terminated, so *lenRead < strlen(filename)
    • lenWritten:uint8_t - the size written in destname. In case of overflows in destname, lenWritten is still increased but destname no longer written. The have the string length in these cases use strlen(destname)
    • terminators:uint8_t - additional characters where a name should terminate. May be NULL
    • Return value:byte - Error code
