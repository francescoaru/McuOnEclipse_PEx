Component SeggerRTT
Segger RTT (Real Time Terminal)
Component Level: High
Category: SW-User Components
Methods:
(Methods are user-callable functions/subroutines intended for the component functions control. Please see the Embedded Components page for more information.)
  • Read - Read from buffer

      ANSIC prototype: int Read(unsigned BufferIndex, const char* pBuffer, unsigned NumBytes)
    • BufferIndex:dword - Buffer/channel to be used. 0 for terminal.
    • pBuffer: Pointer to byte - Pointer to buffer
    • BufferSize:byte - Number of bytes to write
    • Return value:int - Number of bytes that have been read

  • Write - Write to buffer

      ANSIC prototype: int Write(unsigned BufferIndex, char* pBuffer, unsigned BufferSize)
    • BufferIndex:dword - Buffer/channel to be used. 0 for terminal.
    • pBuffer: Pointer to byte - Pointer to buffer
    • BufferSize:byte - Size of buffer
    • Return value:int - Number of bytes which have been written to the up buffer

  • WriteString - Write to buffer

      ANSIC prototype: unsigned WriteString(unsigned BufferIndex, const char* s)
    • BufferIndex:unsigned - Buffer/channel to be used. 0 for terminal.
    • s: Pointer to byte - Pointer to
    • Return value:unsigned - Number of bytes which have been stored in the "Up"-buffer.

  • printf - Stores a formatted string in SEGGER RTT control block. This data is sent to the host.

      ANSIC prototype: int printf(unsigned BufferIndex, const char* sFormat, ...)
    • BufferIndex:unsigned - Index of "Up"-buffer to be used. (e.g. 0 for "Terminal")
    • sFormat:byte - Pointer to format string, followed by the arguments for conversion
    • Return value:int - Error code

  • GetKey - Returns a character/key

      ANSIC prototype: dword GetKey(void)
    • Return value:dword - character code

  • WaitKey - Waits for a key and returns it.

      ANSIC prototype: long WaitKey(void)
    • Return value:long - >=0 Character which has been read.

  • HasKey - Checks if at least one character for reading is available in the SEGGER RTT buffer

      ANSIC prototype: long HasKey(void)
    • Return value:long - 0: No characters are available to read; 1: At least one character is available.

  • SetTerminal - Sets the terminal to be used for output on channel 0.

      ANSIC prototype: int SetTerminal(char TerminalId)
    • TerminalId:byte - Terminal ID, 0..15
    • Return value:int - Error code

  • TerminalOut - Writes a string to the given terminal without changing the terminal for channel 0.

      ANSIC prototype: int TerminalOut(char TerminalId, const char* s)
    • TerminalId:byte - TerminalId, 0..15
    • s: Pointer to uint8_t - Pointer to string
    • Return value:int - Error code

  • ConfigUpBuffer - Configures the Up (device to host) buffer

      ANSIC prototype: int ConfigUpBuffer(unsigned BufferIndex, const char* sName, char* pBuffer, int BufferSize, int Flags)
    • BufferIndex:byte - Buffer index
    • sName:byte - Buffer name
    • pBuffer: Pointer to byte - Pointer to buffer
    • intBufferSize:byte - Size of buffer in bytes
    • Flags:byte - SEGGER_RTT_MODE_NO_BLOCK_SKIP, SEGGER_RTT_MODE_NO_BLOCK_TRIM or SEGGER_RTT_MODE_BLOCK_IF_FIFO_FULL
    • Return value:int - Error code, >=0 OK, <0 Error

  • ConfigDownBuffer - Configures the Down (host to device) buffer

      ANSIC prototype: int ConfigDownBuffer(unsigned BufferIndex, const char* sName, char* pBuffer, int BufferSize, int Flags)
    • BufferIndex:byte - Buffer index
    • sName:byte - Buffer name
    • pBuffer: Pointer to byte - Pointer to buffer
    • intBufferSize:byte - Size of buffer in bytes
    • Flags:byte - SEGGER_RTT_MODE_NO_BLOCK_SKIP, SEGGER_RTT_MODE_NO_BLOCK_TRIM or SEGGER_RTT_MODE_BLOCK_IF_FIFO_FULL
    • Return value:int - Error code, >=0 OK, <0 Error

  • RecvChar - Receives a character from channel 0. Returns ERR_RXEMPTY if no character available

      ANSIC prototype: uint8_t RecvChar(uint8_t *c)
    • c: Pointer to uint8_t - Pointer to where to store the received character
    • Return value:uint8_t - Error code

  • SendChar - Sends a character to channel 0.

      ANSIC prototype: uint8_t SendChar(uint8_t ch)
    • ch:uint8_t - Character to send
    • Return value:uint8_t - Error code. ERR_OK if character has been sent, ERR_TXFULL otherwise.

  • GetCharsInRxBuf - Returns the number of characters in the receive buffer.

      ANSIC prototype: uint16_t GetCharsInRxBuf(void)
    • Return value:uint16_t - Number of characters in the input buffer, zero for none available.

  • StdIOKeyPressed - StdIO handler for Shell

      ANSIC prototype: bool StdIOKeyPressed(void)
    • Return value:bool - True if there are characters in teh input buffer

  • StdIOReadChar - StdIO Handler for reading a character. It returns a zero byte if there is no character in input buffer.

      ANSIC prototype: void StdIOReadChar(uint8_t *c)
    • c: Pointer to uint8_t - Pointer to where to store the received character

  • StdIOSendChar - StdIO handler to sends a character.

      ANSIC prototype: void StdIOSendChar(uint8_t ch)
    • ch:uint8_t - Character to send

  • Init - Initializes the RTT Control Block.

      ANSIC prototype: void Init(void)
