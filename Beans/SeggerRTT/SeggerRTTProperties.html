Component SeggerRTT
Segger RTT (Real Time Terminal)
Component Level: High
Category: SW-User Components
Properties:
(Properties are parameters of the component. Please see the Embedded Components page for more information.)
  • Component name - Name of the component.
  • Version - Version of RTT Library
  • Number of Up Channels - Configures the device-to-host SEGGER_RTT_MAX_NUM_UP_BUFFERS in SEGGER_RTT_Conf.h (size is in bytes, default 1)
  • Number of Down Channels - Configures host-to-device channels with SEGGER_RTT_MAX_NUM_DOWN_BUFFERS in SEGGER_RTT_Conf.h (size is in bytes, default 1)
  • Max Blocked Interrupt Level - Used for SEGGER_RTT_LOCK_INTERRUPT_LEVEL: On Cortex M4 the SEGGER_RTT_LOCK() macro in SEGGER_RTT_Conf.h only blocks interrupts with this interrupt number and below. If FreeRTOS is present, then configMAX_SYSCALL_INTERRUPT_PRIORITY is used.
  • Channel 0 - Settings for buffer 0, typically used for Terminal
    The following items are available only if the group is enabled (the value is "Enabled"):
    • Name - This name is used for the channel. Use "Terminal" for the RTT termiinal view.
    • Up Buffer Size (Tx) - Configures BUFFER_SIZE_UP in SEGGER_RTT_Conf.h (size is in bytes, default 1k)
    • Down Buffer Size (Rx) - Configures BUFFER_SIZE_DOWN in SEGGER_RTT_Conf.h (size is in bytes, default 16)
    • Up Buffer Mode - Mode to be used for Up Buffer

      There are 3 options:
      • Skip (Default): Skip. Do not block, output nothing. (Default)
      • Trim: Trim: Do not block, output as much as fits.
      • Block: Block: Wait until there is space in the buffer.

    • Down Buffer Mode - Mode to be used for Down Buffer

      There are 3 options:
      • Skip (Default): Skip. Do not block, output nothing. (Default)
      • Trim: Trim: Do not block, output as much as fits.
      • Block: Block: Wait until there is space in the buffer.

    • Blocking Send - Affects if StdIOSendChar shall be blocking or not
      The following items are available only if the group is enabled (the value is "Enabled"):
      • Timeout (ms) - Timeout, how long to wait for a single character. Use zero for blocking send.
      • Wait - Interface to wait routines
      • Wait Time (ms) - Waiting time in milli seconds if sending was blocked (output buffer full). Use 0 for polling mode.
    • Printf Buffer Size - Configures SEGGER_RTT_PRINTF_BUFFER_SIZE in SEGGER_RTT_Conf.h (size is in bytes, default 64)
  • SDK - Link to SDK
